# Article Editing

文章文字校对与编辑 skill。

## 用途

对已存在的文章进行文字校对、错字修正、病句修改等编辑工作。

## 使用方法

当用户请求校对、修改或编辑某篇文章时使用此 skill。

## 编辑流程

### 1. 确定目标文章

询问用户要编辑的文章名称或直接读取用户指定的文章。

### 2. 读取文章内容

读取 `draft/posts/{文章名}.md` 文件。

### 3. 进行文字校对

#### 常见错误类型

- **错别字**：如"的"与"得"、"地"的混淆
- **标点错误**：全角/半角混用、缺少标点、错误使用标点
- **语法错误**：主谓搭配不当、语序混乱、成分残缺
- **用词不当**：近义词混淆、词义重复
- **格式问题**：标题层级混乱、列表格式不统一

#### 常见检查点

1. **人称一致性**：全文使用统一的人称
2. **时态一致性**：保持时态统一
3. **繁简统一**：全篇使用繁体或简体，不可混用
4. **术语统一**：同一术语在全文中保持一致
5. **标题格式**：同级别标题格式统一

### 4. 提供修改建议

将校对结果呈现给用户，包括：
- 错误位置（行号）
- 错误类型
- 原文
- 修改建议

### 5. 应用修改

在用户确认后，使用 Edit 工具对文章进行修改。

## 常见错误示例

### 错别字

```
原文：我想像以前一样生活
改文：我想像以前一样生活 ✓

原文：即然你已经决定了
改文：既然你已经决定了 ✓
```

### 标点问题

```
原文：今天天气很好。（虽然有点冷）
改文：今天天气很好，虽然有点冷。✓

原文：他说"你好"
改文：他说："你好" ✓
```

### 病句

```
原文：通过这次活动，让我们明白了团结的重要性。
改文：通过这次活动，我们明白了团结的重要性。✓

原文：这本书的内容和有趣。
改文：这本书的内容很有趣。✓
```

## 编辑原则

1. **最小改动原则**：在保持原意的前提下，尽量减少改动
2. **尊重作者意图**：如果不确定，宁可保留原样
3. **格式统一**：确保标题、列表、引用等格式统一
4. **保留专业术语**：对于领域专用术语，保持原有写法

## 批量校对

如果需要校对多篇文章，可以：

1. 列出所有文章
2. 逐篇进行校对
3. 汇总所有修改建议后统一提交
